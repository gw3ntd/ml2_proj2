## cifarcomp

- Using the functions from functions.py on a randomly generated dataset

## customlayer
- Has the code for my custom layer

## functions
Has three functions:
1. **get_keras_vals**: gets Keras's LayerNormalization outputs for a given tensor
2. **get_custom_vals**: gets my custom LayerNormalization layer outputs for a given tensor
3. **comp_layer**: compares the outputs from the previous two functions based on true difference, l2 norm, and l infinity norm
